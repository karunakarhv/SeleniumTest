Pre-Requisites to setup on your laptop/pc:
1. Python 3 Installed and configured in the System Path
2. Not really required - you can install virtual environment if packages needed to be separated with your existing repository.
3. pip install selenium
4. Download Firefox Web Driver(geckodriver) from the firefox releases website
   https://github.com/mozilla/geckodriver/releases
5. Download and Install VSCode

To Code and Run:
1. Code
2. To Run we can run in terminal python3 RacingPage.py or Ctrl-F5
